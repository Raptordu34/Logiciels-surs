Exercice 4 : Multiplication égyptienne

Énoncé

L'algorithme proposé effectue la multiplication a * b en utilisant des additions, des multiplications par 2 et des divisions par 2.

x := a;
y := b;
total := 0;
while x > 0 do
  if x mod 2 = 1 then
    total := total + y ; x := x - 1;
  end;
  x := x/2;
  y := y*2;
  INVARIANT x ∈ IN ∧ total + x * y = a * b
  VARIANT x
end
Postcondition : total = a * b

Prouver la correction de cet algorithme (Initialisation, Préservation, Terminaison).

Solution

1. Initialisation

Avant la boucle, on a x = a, y = b, et total = 0.

On remplace dans l'invariant : 0 + a * b = a * b. L'égalité est vraie, donc l'invariant est établi au démarrage de la boucle.

2. Préservation de l'invariant

Supposons l'invariant vrai au début d'une itération. On distingue deux cas selon la parité de x :

  • Cas où x est impair (x mod 2 = 1) :
    Nouvelles valeurs en fin d'itération : x' = (x-1)/2, y' = y * 2, total' = total + y.
    Vérifions : total' + x' * y' = (total + y) + ((x-1)/2) * (y * 2) = total + y + (x-1)y = total + y + xy - y = total + xy = a * b.
    L'invariant est préservé.
  • Cas où x est pair (x mod 2 = 0) :
    Nouvelles valeurs : x' = x/2, y' = y * 2, total' = total.
    Vérifions : total' + x' * y' = total + (x/2) * (y * 2) = total + xy = a * b.
    L'invariant est également préservé.

3. Terminaison et Postcondition

Le variant V = x décroît strictement car à chaque itération x devient soit x/2 soit (x-1)/2. Puisque la boucle tourne tant que x > 0, la valeur finira obligatoirement par atteindre 0.

En sortie de boucle, la condition x > 0 est fausse. Sachant par l'invariant que x ∈ IN, cela implique que x = 0.

Si on remplace x = 0 dans l'invariant : total + 0 * y = a * b &implies; total = a * b. La postcondition est prouvée avec succès !

Ceci illustre la démarche complète des Obligations de Preuve (OP) en méthode B : on démontre mathématiquement que l'initialisation établit l'invariant, que chaque opération le préserve, et que le système atteint bien le but fixé (postcondition).

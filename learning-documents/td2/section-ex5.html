Exercice 5 : Feux de circulation

Énoncé

Écrire une Machine Abstraite B pour modéliser un carrefour avec des feux de circulation (directions Est-Ouest et Nord-Sud).

Propriétés à respecter :

  • Sûreté : Si une direction est Vert ou Orange, l'autre est obligatoirement Rouge.
  • Séquentialité : L'ordre des couleurs est Rouge → Vert → Orange → Rouge.
Rappel de cours : La machine abstraite est un contrat. L'INVARIANT décrit les propriétés de sûreté intemporelles (ex: pas de collision), tandis que les PRE (préconditions) des opérations régissent la dynamique (séquentialité et permissions).

Solution : Machine Abstraite B

Voici la modélisation complète de la machine :

MACHINE
  FeuxCirculation

SETS
  COULEUR = {Rouge, Vert, Orange};
  DIRECTION = {EstOuest, NordSud}

VARIABLES
  feux

INVARIANT
  /* feux est une fonction totale : à chaque direction correspond exactement une couleur */
  feux : DIRECTION --> COULEUR &
  /* Propriété de Sûreté (pas de collision possible) */
  (feux(EstOuest) : {Vert, Orange} => feux(NordSud) = Rouge) &
  (feux(NordSud) : {Vert, Orange} => feux(EstOuest) = Rouge)

INITIALISATION
  /* Sécurité maximale au démarrage : tout au Rouge */
  feux := DIRECTION * {Rouge}

OPERATIONS

  PasseAuVert(dir) =
  PRE
    dir : DIRECTION &
    /* Séquentialité */
    feux(dir) = Rouge &
    /* Sûreté : on ne passe au vert que si l'autre est rouge */
    (dir = EstOuest => feux(NordSud) = Rouge) &
    (dir = NordSud => feux(EstOuest) = Rouge)
  THEN
    feux(dir) := Vert
  END;

  PasseAlOrange(dir) =
  PRE
    dir : DIRECTION &
    /* Séquentialité */
    feux(dir) = Vert
  THEN
    feux(dir) := Orange
  END;

  PasseAuRouge(dir) =
  PRE
    dir : DIRECTION &
    /* Séquentialité */
    feux(dir) = Orange
  THEN
    feux(dir) := Rouge
  END

END

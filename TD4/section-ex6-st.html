Exercice 6 — Vérification sur un système de transitions

Intermédiaire ◎ 12 pts ⏱ 20–25 min
Objectifs
  • Vérifier des formules LTL sur un système de transitions (Kripke)
  • Analyser tous les chemins d'un ST pour valider ou réfuter une formule
  • Trouver un chemin contre-exemple pour réfuter □φ

Prérequis : Cours — Systèmes de transitions, modèle de Kripke, sémantique LTL sur ST

Système de transitions (ST)

On considère le système de transitions suivant :

Topologie du ST

États et étiquettes :
  S1 {p, q}   ← état initial
  S2 {p, q}
  S3 {p}

Transitions :
  S1 → S2     S1 → S3
  S2 → S2     S2 → S3
  S3 → S1

Lecture : S1 est l'état initial. Les propositions atomiques vraies sont listées entre accolades. Une flèche S → S' indique une transition possible.

ÉtatpqSuccesseurs
S1 (initial)✓✓S2, S3
S2✓✓S2, S3
S3✓✗S1

ST ⊨ φ signifie que φ est vraie sur tous les chemins infinis du système depuis l'état initial.

Pour réfuter une formule : trouver un seul chemin infini depuis S1 sur lequel φ est fausse.

Pour valider une formule : prouver qu'elle est vraie sur tous les chemins possibles.

Méthode

  • Lister les chemins possibles (en tenant compte des boucles).
  • Pour chaque formule □φ : chercher un état atteignable où φ est fausse.
  • Pour φ U ψ : vérifier si ψ finit par devenir vrai sur le chemin considéré.

Chemins depuis S1 : Voici les chemins représentatifs (S2 peut se boucler plusieurs fois) :

  • S1 → S2 → S2 → S2 → … (boucle infinie en S2)
  • S1 → S2 → S3 → S1 → S2 → … (cycle S2→S3→S1)
  • S1 → S3 → S1 → S3 → … (cycle S1↔S3)
  • S1 → S3 → S1 → S2 → S3 → … (mixte)
Q1 3 pts Fondamental

Vérifier si ST ⊨ □p.

Piège courant : Vérifier seulement l'état initial S1. □p exige que p soit vrai dans tous les états atteignables sur tous les chemins infinis.
Regardez la colonne "p" dans le tableau des états. Tous les états ont-ils p = ✓ ?

Solution — Q1

ST ⊨ □p est VRAIE.

p ∈ {p,q} = étiquette de S1 ✓
p ∈ {p,q} = étiquette de S2 ✓
p ∈ {p} = étiquette de S3 ✓

Tous les états atteignables ont p = vrai. Sur tout chemin infini depuis S1, p est vrai à chaque position. Donc □p est satisfaite sur tous les chemins, et ST ⊨ □p.

Q2 3 pts Intermédiaire

Vérifier si ST ⊨ □(p ∧ q).

Piège courant : Confondre □p (vraie ici) avec □(p∧q). La conjonction exige que les deux propositions soient vraies simultanément à chaque état.
Y a-t-il un état atteignable où p∧q est fausse ? Regardez l'étiquette de S3.

Solution — Q2

ST ⊨ □(p∧q) est FAUSSE.

Contre-exemple : le chemin S1 → S3 → S1 → …

En position 1 (état S3) : étiquette = {p}. q ∉ {p}, donc p∧q est fausse en S3. Ce chemin viole □(p∧q). Donc ST ⊭ □(p∧q).

Q3 3 pts Intermédiaire

Vérifier si ST ⊨ □(p → ¬q).

Piège courant : Évaluer seulement S3 (où p∧¬q est vrai) et conclure que la formule tient. Il faut vérifier tous les états. En S1 et S2, p est vrai — que vaut ¬q là ?
En S1 : p = vrai, q = vrai. Donc p → ¬q = vrai → faux = faux. Cela suffit pour réfuter la formule.

Solution — Q3

ST ⊨ □(p → ¬q) est FAUSSE.

Contre-exemple : tout chemin partant de S1.

En position 0 (état initial S1) : p = vrai, q = vrai.
Donc p → ¬q = vrai → faux = faux.
□(p→¬q) est fausse dès le premier état. ST ⊭ □(p → ¬q).

Q4 3 pts Avancé

Vérifier si ST ⊨ q U (p → ¬q).

Piège courant : Chercher un instant où p → ¬q est vraie globalement plutôt que sur un chemin précis. Until s'évalue sur un chemin : q doit tenir à tous les instants avant que p→¬q ne devienne vrai.
q U ψ : chercher un instant k où ψ est vrai, tel que q est vrai à tous les instants 0, 1, …, k-1. Quelle est la valeur de p→¬q en S3 ?
En S3 : p = vrai, q = faux. Donc p→¬q = vrai→vrai = vrai. Sur le chemin S1 → S2 → S3 → … : q est vrai en S1 et S2 (positions 0 et 1), et p→¬q est vrai en S3 (position 2).

Solution — Q4

ST ⊨ q U (p → ¬q) est FAUSSE (avec Until fort).

Pour que q U ψ soit vraie, il faut que sur tout chemin, ψ devienne vrai à un instant k, et que q soit vrai avant.

Contre-exemple : le chemin infini qui boucle sur S2 : S1 → S2 → S2 → S2 → …

  • En S1 et S2, q est vrai. ✓
  • Mais en S1 et S2, ψ = (p → ¬q) est faux (vrai → faux).
  • Sur ce chemin, ψ ne devient jamais vrai.

Note sur la sémantique :

  • Until fort (U) : ψ doit obligatoirement devenir vrai. Ici, le chemin restant bloqué en S2 invalide la formule.
  • Until faible (W) : la formule serait vraie car q reste vrai indéfiniment.

Dans la plupart des exercices de vérification formelle, on utilise le Until fort par défaut.

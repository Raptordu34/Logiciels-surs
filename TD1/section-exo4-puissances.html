Exercice 4 — Puissances d'une relation

Intermédiaire ◎ 8 pts ⏱ 20–30 min
Objectifs
  • Calculer la composition de deux relations par un procédé systématique
  • Calculer les puissances successives d'une relation
  • Identifier un cycle dans la suite des puissances et l'exploiter

Prérequis : Cours 2 — Composition et puissances de relations

Contexte — Puissances d'une relation

Soit la relation R définie par :

R = { (0 → 2), (1 → 0), (2 → 1) }

Calculer R¹, R², R³ et R⁷.

Rappel de notation
  • R¹ = R
  • R² = R ; R  (R composé avec lui-même)
  • Rⁿ = Rⁿ⁻¹ ; R  pour n ≥ 2
  • Composition S ; R : (x, z) ∈ S ; R  ⟺  ∃ y · (x, y) ∈ S ∧ (y, z) ∈ R

La composition S ; R produit toutes les paires (x, z) telles qu'il existe un intermédiaire y avec (x, y) ∈ S et (y, z) ∈ R.

Pour calculer R², on cherche, pour chaque paire (x, y) ∈ R, une paire (y, z) ∈ R ayant le même intermédiaire y, et on produit (x, z).

Méthode systématique pour calculer S ; R :

  • Pour chaque paire (x, y) ∈ S, chercher toutes les paires (y, z) ∈ R ayant y comme antécédent.
  • Chaque paire (y, z) trouvée produit une nouvelle paire (x, z) dans le résultat.
  • Si aucune paire de R ne commence par y, la paire (x, y) de S ne contribue à rien.

Point clé

La composition n'est pas commutative en général : S ; R ≠ R ; S. Dans cet exercice, c'est toujours R qui joue le rôle du côté droit — vérifiez l'ordre à chaque étape.

Exemple — composition

Soit P = { a → b, b → c } et Q = { b → x, c → y }.

P ; Q :

  • (a → b) ∈ P et (b → x) ∈ Q → produit (a → x)
  • (b → c) ∈ P et (c → y) ∈ Q → produit (b → y)

P ; Q = { a → x, b → y }

Analogie

Intuition : composer une relation, c'est enchaîner deux étapes d'un itinéraire. Si on va de x à y en une étape (S), et de y à z en une étape (R), alors S ; R dit qu'on peut aller de x à z en deux étapes.

Les puissances Rⁿ représentent les destinations accessibles en exactement n étapes dans le graphe de R.

Q1 5 pts Intermédiaire

Soit la relation :

R = { (0 → 2), (1 → 0), (2 → 1) }

Calculez R¹, R² et R³. Détaillez chaque étape de composition.

Piège courant : Confondre la composition avec l'union. R² = R ; R ne contient que les paires (x, z) reliées via un intermédiaire dans R — ce n'est pas R ∪ R ni un simple "doublement" des éléments.
Pour R² = R ; R, prenez chaque paire (x → y) ∈ R et cherchez (y → z) ∈ R. Par exemple : (0 → 2) ∈ R — cherchez une paire débutant par 2 dans R : (2 → 1). Cela donne (0 → 1) ∈ R².
R² = { 0 → 1, 1 → 2, 2 → 0 }. Pour R³ = R² ; R, refaites le même procédé en prenant R² à gauche et R à droite — cherchez les intermédiaires dans R.

Solution — Q1

R¹ = R (par définition)

R¹ = { 0 → 2, 1 → 0, 2 → 1 }

R² = R ; R — pour chaque (x → y) ∈ R, chercher (y → z) ∈ R :

Paire dans RIntermédiaire yPaire (y → z) dans RRésultat (x → z)
0 → 2y = 22 → 10 → 1
1 → 0y = 00 → 21 → 2
2 → 1y = 11 → 02 → 0
R² = { 0 → 1, 1 → 2, 2 → 0 }

R³ = R² ; R — même procédé avec R² à gauche :

Paire dans R²Intermédiaire yPaire (y → z) dans RRésultat (x → z)
0 → 1y = 11 → 00 → 0
1 → 2y = 22 → 11 → 1
2 → 0y = 00 → 22 → 2
R³ = { 0 → 0, 1 → 1, 2 → 2 } = Id (relation d'identité sur {0, 1, 2})

R³ est la relation d'identité : chaque élément est mis en correspondance avec lui-même. Ce résultat remarquable n'est pas une coïncidence — voir Q2.

Q2 3 pts Avancé

Calculez R⁷ sans effectuer toutes les compositions intermédiaires. Justifiez le raisonnement.

Piège courant : Calculer R⁴, R⁵, R⁶, R⁷ une par une. Ce procédé est correct mais inutilement long — le résultat de Q1 contient déjà le raccourci nécessaire.
R³ = Id (l'identité). Id ; R = R pour toute relation R. Que vaut donc R⁴ = R³ ; R ?
R⁴ = R¹ = R. La suite est périodique de période 3 : R¹ = R⁴ = R⁷, R² = R⁵, R³ = R⁶ = Id. Calculez 7 mod 3 pour identifier à quelle puissance "de base" R⁷ correspond.

Méthode

  • Identifier si une puissance de R est une relation connue (Id, R, R², ∅…).
  • En déduire la période du cycle.
  • Réduire l'exposant cible modulo la période.
  • Conclure par identification avec la puissance de base correspondante.

Solution — Q2

  • On a montré que R³ = Id.
  • Donc R⁴ = R³ ; R = Id ; R = R = R¹.
  • De même : R⁵ = R², R⁶ = R³ = Id, R⁷ = R⁴ = R¹.
  • La suite (Rⁿ) est périodique de période 3 : Rⁿ = Rn mod 3.
  • 7 mod 3 = 1, donc R⁷ = R¹ = R.
R⁷ = R = { 0 → 2, 1 → 0, 2 → 1 }
Pourquoi R est-elle cyclique de période 3 ?

R est une permutation circulaire de {0, 1, 2} : elle envoie 0 sur 2, 2 sur 1, 1 sur 0 — une rotation d'ordre 3. Toute permutation d'ordre k vérifie Rᵏ = Id, ce qui engendre le cycle observé ici.

Pour aller plus loin
  • Quelle est la relation R⁰ ? Donnez sa définition formelle et vérifiez qu'elle est cohérente avec la formule Rⁿ = Rⁿ⁻¹ ; R pour n = 1.
  • Si on modifie R en ajoutant la paire (0 → 0), la suite des puissances reste-t-elle périodique ? Quel est l'effet sur R² et R³ ?
  • Montrez formellement que si Rᵏ = Id pour une permutation R, alors Rⁿ = Rn mod k pour tout n ≥ 0.

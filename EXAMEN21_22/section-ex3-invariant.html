Exercice 3 — Invariant et Opérations

Intermédiaire ◎ 3 pts ⏱ 15–20 min
Objectifs
  • Évaluer une substitution parallèle dans le formalisme B
  • Vérifier la préservation d'un invariant par une opération

Prérequis : Cours — Substitutions généralisées en B, invariant de machine

Substitution parallèle (||) : toutes les affectations sont évaluées simultanément avec les valeurs avant l'exécution. L'ordre d'écriture n'a pas d'importance.

Préservation d'invariant : une opération préserve l'invariant I si, pour tout état satisfaisant I et la précondition PRE, l'état après exécution satisfait encore I.

Formellement

I ∧ PRE ⇒ [S]I — où [S]I est la plus faible précondition de I par la substitution S.

Q1 1.5 pt Fondamental

Donner le résultat de l'opération suivante pour l'appel op(2, 1) avec v1 = 1 et v2 = 4.

sortie ← op(x, y) =
PRE x ∈ ℕ ∧ y ∈ ℕ
THEN
    v1 := v1 + x ||
    v2 := v2 + y ||
    sortie := v1 * v2
END
Piège courant : La substitution est parallèle (||). L'affectation sortie := v1 * v2 utilise les valeurs de v1 et v2 avant modification, pas les nouvelles valeurs.
Avec la substitution parallèle, calculez d'abord toutes les expressions avec les valeurs initiales (v1=1, v2=4), puis affectez les résultats simultanément.

Solution — Q1

Les trois affectations sont évaluées simultanément avec les valeurs initiales v1 = 1, v2 = 4 :

VariableExpressionCalculRésultat
v1v1 + x1 + 23
v2v2 + y4 + 15
sortiev1 * v21 × 44

Résultat : v1 = 3, v2 = 5, sortie = 4.

Q2 1.5 pt Intermédiaire

Déterminer si l'opération suivante préserve l'invariant y ∈ {1, 2, 3, 4, 5}. Justifier.

op(x) = PRE x ∈ {1, 2} ∧ y ∈ {1, 2} THEN y := y + (2 * x) END
Piège courant : Il ne suffit pas de tester un seul cas. Il faut vérifier toutes les combinaisons possibles de x et y autorisées par la précondition.
Énumérez toutes les combinaisons : x ∈ {1, 2} et y ∈ {1, 2}, soit 4 cas au total.
Calculez y + 2x pour chaque combinaison et vérifiez si le résultat appartient toujours à {1, 2, 3, 4, 5}.

Solution — Q2

L'invariant n'est PAS préservé.

Énumération de tous les cas :

Pour x = 2 et y = 2 : y' = 2 + 4 = 6, qui n'appartient pas à {1, 2, 3, 4, 5}.

L'invariant est donc violé par ce contre-exemple.
